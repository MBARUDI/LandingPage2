# Landing Page - FonoClínica (Dr. Barudi)

Uma Landing Page moderna, responsiva e de alta conversão desenvolvida para uma clínica de fonoaudiologia, focada no **Dr. Barudi**. O projeto utiliza as tendências de design mais atuais de 2026, com foco em acolhimento, confiança e profissionalismo.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **Tailwind CSS (via CDN)**: Estilização rápida e moderna com foco em performance e design premium.
- **Google Fonts (Outfit)**: Tipografia moderna e altamente legível.
- **Phosphor Icons**: Biblioteca de ícones vetoriais elegantes e consistentes.
- **Design Dinâmico**: Uso de Glassmorphism, gradientes e animações orgânicas (blobs).

## 📋 Seções da Página

1.  **Hero**: Introdução impactante com título sobre comunicação e bem-estar, chamada para ação (CTA) e elementos visuais dinâmicos.
2.  **Serviços/Especialidades**: Cards informativos sobre Linguagem Infantil, Voz Profissional e Reabilitação Auditiva.
3.  **Sobre**: Biografia profissional do Dr. Barudi, focada em sua experiência e abordagem humanizada.
4.  **Depoimentos**: Seção em estilo *Bento Grid* com relatos reais de pacientes, reforçando a prova social.
5.  **Rodapé**: Informações de contato (WhatsApp, E-mail, Endereço) e links para redes sociais.

## 🛠️ Como Executar o Projeto

Como o projeto utiliza Tailwind CSS via CDN e fontes externas, não é necessário realizar instalações complexas.

1.  Clone o repositório:
    ```bash
    git clone https://github.com/MBARUDI/LandingPage2.git
    ```
2.  Navegue até a pasta do projeto e abra o arquivo `index.html` em qualquer navegador moderno.

## 🌐 Deploy (GitHub Pages)

Este projeto está pronto para ser hospedado via GitHub Pages. Para ativar:
1. Vá em **Settings** > **Pages**.
2. Selecione a branch `main` e a pasta `/root`.
3. Salve e aguarde o link de publicação.
